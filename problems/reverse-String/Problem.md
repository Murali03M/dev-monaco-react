## Reverse String

Reverse the given string.

### Problem Statement

Given a string, write a function to reverse the string and return the reversed version.

#### Input
- A single string.

#### Output
- The reversed version of the input string.



#### Test case 1

Input

```
hello
```

Output

```
olleh
```

#### Test case 2

Input

```
world
```

Output

```
dlrow
```
