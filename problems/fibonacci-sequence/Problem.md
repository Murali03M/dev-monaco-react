### Problem Statement

Write a function that takes a non-negative integer \( n \) as input and returns the \( n \)-th Fibonacci number.

#### Input
- A single integer \( n \) where \( 0 \leq n \leq 30 \).

#### Output
- The \( n \)-th Fibonacci number.


#### Test case 1

Input

```
5
```

Output

```
5
```


#### Test case 2

Input

```
10
```

Output

```
55
```
